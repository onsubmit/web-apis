The Web APIs extend the funtionality of the browser. Many are experimental and/or partially supported by the various browser vendors. Be cautious when using them on your Web app or site, ensuring your targeted browsers fully support them.

This documention is intended to complement MDN, providing interactive examples that run directly in the browser.
